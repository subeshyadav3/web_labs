# JavaScript Lab Assignments (Vanilla JS)

This repository contains a collection of **JavaScript laboratory assignments** completed as part of academic coursework.  
All assignments are implemented using **Vanilla JavaScript**, focusing on core language concepts, DOM manipulation, and event-driven programming without external libraries or frameworks.

Each assignment follows a **standard lab report format** as required.

---

## üìå General Lab Report Structure

For every assignment, the following sections are included:

1. **Problem Understanding**  
   Clear interpretation of the given problem statement and requirements.

2. **List of JavaScript Concepts Used**  
   Core JavaScript features, DOM APIs, events, arrays, objects, timers, etc.

3. **Flow Diagram / Logic Steps**  
   Step-by-step explanation of program logic or algorithmic flow.

4. **Source Code**  
   Complete implementation using Vanilla JavaScript, HTML, and CSS.

5. **Output Screenshots**  
   Screenshots demonstrating correct functionality and UI output.

### **Assignment 1: Smart Calculator with History**
**Features:**
- Performs basic arithmetic operations
- Stores last 5 calculations using an array
- Dynamically displays calculation history
- Option to clear history
- Handles invalid inputs (e.g., divide by zero)

### **Assignment 2: Student Marks & Grade Analyzer**
**Features:**
- Stores student details as objects
- Accepts marks for multiple subjects
- Calculates total, percentage, and grade
- Displays results in a table
- Highlights pass/fail rows using CSS via JavaScript

### **Assignment 3: Dynamic To-Do List with Status Filter**
**Features:**
- Add, delete, and mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Display count of completed tasks

**Features:**
- Questions stored as objects

**Features:**

**Features:**
- Accepts number of rows and columns as user input
- Dynamically generates a table
